# Briefs SEO par page — White & Care

Un document par landing page, produit après analyse concurrentielle (cabinets
dentaires Anderlecht / Bruxelles) et best practices SEO local 2026. Chaque brief
contient du **contenu rédigé prêt à coller** : title/meta, structure Hn, paragraphes,
FAQ (à baliser en `FAQPage`), maillage interne, SEO local et mots-clés.

> Étape suivante : intégrer ces contenus dans `src/pages.mjs` puis `node build.mjs`.
> Travailler page par page, valider les FAQ sur le Rich Results Test.

## Garde-fous appliqués dans tous les briefs

- **Conventionnement** : « une grande équipe de dentistes conventionnés » — jamais
  « 100% / tous / sans exception / sur demande ». **Zéro** mention conventionné/INAMI/
  remboursé sur les pages spécialistes (implant, Invisalign, orthodontie).
- Français de Belgique, vouvoiement, ton rassurant. Aucun tiret long, aucun emoji.
- Action unique = **appeler** (`tel:+32 2 493 53 53`).
- Contenu 100 % original (les concurrents servent de référence de structure, pas de copie).

## À valider avec le cabinet avant publication

- Chiffres de remboursement INAMI / fourchettes de prix (formulés prudemment, à confirmer).
- Existence de créneaux d'urgence quotidiens / modalités de garde le week-end.
- Accessibilité PMR, harmonisation NAP (code postal 1070).
